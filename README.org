* Summary(<2016-12-07 Wed>)
1. 现在主要关注的是对抗网络和RNN， 感觉这也是以后发展的重点，最近听了一部分的拓扑数据分析的课程，感觉还是有一些用处的，但是具体的将拓扑数据分析和现在的深度学习结合感觉没有那么多的契合点。
